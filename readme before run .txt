Voici la version final de mon projet de fin de formation j'espère que vous aimerez =)

Il s'agit d'un site vitrine avec une petite boutique et un système de panier et un systeme de commande (il est nécéssaire d'être identifié pour valider la commande).

Pour la page Home le slider est en dynamique, idem pour la page prestations et boutique ou tout est dynamique. Pour la page contact le formulaire est enregistré dans la bdd également et la carte est faite via l'api leaflet.

Il y a un dashboard admin (action= loginAdmin, je vous laisse crée un admin via le fichier de config createAdmin (bien pensez à remplir les strings vides) sinon vous pouvez utiliser le mien:
-enguerrant@gmail.com
-admin) 
ou vous pouvez gérer l'intégralité du site ( slider, shop, prestation , et voir et télécharger les formulaires de contact)

- Il y a un petit espace client ou vous pourrez voir vos commandes et modifier quelques informations personnels et télécharger vos factures (attention à bien respecter les formats mail, téléphone et mot de passe demandé sinon vous serez alerté par une erreur lors de la soumission du formulaire)

Le CSS a été réalisé avec SASS, j'ai utilisé du javacpript natif ainsi que du Jquerry. 

Le site à été réalisé sous une architecture MVC et avec de la POO.

Vous trouverez la base de donnée si vous voulez testé en local dans www/sql (dans ce cas bien pensé a modifier les informations de connexion à la BDD dans le fichier de configuration à la BDD)

Je vous souhaite une bonne visite et un bon test de fonctionnalité et n'oublions jamais :" VIVE HELLO WORLD" :D .
